# Personal Portfolio Website


## Overview
This is my personal portfolio website built using HTML, CSS, and JavaScript.  
It showcases my skills, projects, and contact information in a clean, responsive, and modern dark-mode design.

The website features:
- Dark mode UI for modern and eye-friendly look
- Responsive design (works on mobile, tablet, and desktop)
- Section-based navigation – only one section visible at a time
- Project cards highlighting key projects
- Skills list with professional styling



##  Technologies Used
- HTML5 – Semantic markup
- CSS3 – Flexbox, Grid, hover effects, dark mode
- JavaScript – Section-based navigation (dynamic show/hide)



##  Project Structure
personal-portfolio/
├── index.html # Main HTML file
├── style.css # Dark mode & responsive CSS
├── script.js # Section navigation JS
└── README.md # Project documentation


##  Features
- Home section with welcome message
- About section with bio
- Skills section highlighting technical skills
- Projects section showing key projects
- Contact section with email & LinkedIn link
- Fully responsive and clean dark mode UI
- Smooth section navigation using JavaScript
